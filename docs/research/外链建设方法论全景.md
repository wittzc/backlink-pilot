# 外链建设方法论全景

> **角色**：外链建设领域的**全景方法论知识库**，给 backlink-pilot 的产品决策（哪些战术值得做成功能、哪些明确不做）当输入，不是用户使用手册。来源是 [web.cafe「外链建设」标签](https://new.web.cafe/label/b8a3104c000d448b9ebf3c8e86dfffdd)下 28 篇帖子（哥飞 + 社群高手）的精读提炼，2026-06-11 整理。
> **与 [METHODOLOGY.md](../METHODOLOGY.md) 的关系**：本文是「领域里有哪些打法」的全集，METHODOLOGY 是「本产品做什么 / 不做什么」的边界。本文记录的部分战术（如 nofollow 绕过、过期域名 301）**工具不实现**——backlink-pilot 遵循 METHODOLOGY 的 fail-fast 边界，这些仅作领域认知与方向参考。

## 一句话策略

外链只有两个目的：**提升权重**（长期 SEO，看不出 ROI 但持续起效）和**获取流量**（短期，能算 ROI）。先分清你在做哪个，再决定怎么发。
拿权重的最高效路径不是自己瞎找，而是**抄已经排上去的同行**：把竞品的外链域名拉出来按出现次数排序，重合度高的就是你也能拿到的。配上「一天 1-2 条、锚文本多样化、不贪多」的节奏，1.5-2 个月能看到起量。

## 一、先搞懂：外链到底在干什么（认知层）

动手前必须理解的 5 条机制，否则发再多也排不上去。

### 1. 权重 vs 流量，两种外链别混

|     | 提升权重型                       | 获取流量型                    |
| --- | --------------------------- | ------------------------ |
| 目的  | 抬高网站权威度 → 抬关键词排名 → 拿 SEO 流量 | 直接从发布处带访客来注册/付费          |
| 时间  | 长期，链接存在多年持续起效               | 短期，链接撤下就没了               |
| ROI | **算不出来**，别用 ROI 衡量          | 能算（类似广告投放）               |
| 例子  | 目录站收录、博客评论、客座博客             | 社区发帖、Product Hunt 流量、广告位 |

> 群里说的「做外链」默认指**第一种**。别因为「发出去没人点」就觉得没用，权重是慢慢沉淀的。（帖 12，哥飞）

### 2. 链接价值取决于「谁链你」，不是数量

一个权威且相关的域名链你，胜过一百个无关小博客。搜索引擎看的是给链者本身的权威度和主题相关性。这是 PageRank / Domain Authority 的底层逻辑。（帖 04，哥飞「链接寓言」）

### 3. 权重有时间维度，暴增 = 找死

信任是慢慢建立的。短时间外链暴增会被识别为操纵（link scheme）触发惩罚。老域名的权威很难被新域名短期超越。所以**节奏比数量重要**。（帖 04）

### 4. 最强的链接是副产品，不是刻意建的

Earned links（别人自发链你）> Built links（你主动去发的）。前者是 Google 眼里最高权重的信号。做出「免费而丰富的高价值资源」让人自发传播，效果远超手动发一万条。（帖 04 / 帖 07）

### 5. 首页 vs 内页：为什么有些站用内页拿排名

- 首页权重最高（外链/内链都汇聚过来），既是结果也是原因。
- 但 Google 的惩罚粒度多是**内页**而非整站。所以有站长故意**用随机内页承接排名**（`/` 跳转到 `/en-XXXX`，用 [wheregoes.com](https://wheregoes.com) 可检测）：内页被罚就再开一个新内页，首页权重不受损；首页若被罚则整站废掉。
- 这也解释了「为什么有些词流量大但 Ahrefs KD 很低」：Ahrefs 认为搜索结果里首页越多越难，而这些站几乎不用首页拿排名，算法盲区导致 KD 算得极低（甚至 4 个外链就能进首页）。
- **反推**：用关键词注册域名，举全站之力优化，**用你的首页去打别人的内页**，Google 认为你更专业、更偏爱小而美的站。看 KD 别只看数值，综合流量和外链数判断。（帖 21，哥飞）

## 二、核心心法：抄同行外链（哥飞 SOP）

这是整个标签里被反复验证、最值钱的一条。可复制的标准流程：

1. 在 [toolify.ai](https://www.toolify.ai) 和 [theresanaiforthat.com](https://theresanaiforthat.com) 找跟你产品类似的产品，**尽可能多找**。
2. 用 [Ahrefs free backlink checker](https://ahrefs.com/backlink-checker/) 或 Semrush，**逐个查这些产品的外链域名列表**。
3. 把所有外链域名**汇总，按出现次数从高到低排序**。出现次数越高 = 越多同行都能拿到 = 你大概率也能拿到。这就是一份「绝大多数网站都能拿到的外链列表」。
4. 按这个列表逐个去发。

> 为什么有效：能在几年内排上去的新站，靠的就是可复制的外链。它能拿的，你也能拿。（帖 25 SOP 原文 + 帖 19）

### 日常顺手收集法（边找词边攒外链）

把收集外链塞进日常找词流程，几乎零额外成本：

- Google 搜任意目标关键词，看搜索结果**前 3 名**（竞争激烈的词看前 2 页）。
- 凡是域名**近 5 年注册**的，就记下它的外链。逻辑同上：新站排上去靠的是可复制外链。
- **不看流量大小**，只看「排名位置 + 域名年龄」两个信号。
- 一周攒的就够发了，后续按 DR 从高到低筛。

工具：[AITDK](https://aitdk.com)（看域名注册时间 + 流量）、Ahrefs backlink checker + bookmarklet（见第六部分）。（帖 19，安于不安）

## 三、找外链源的 6 条路径

抄同行是主路径，这 6 条是它的展开和补充。

### 1. 竞品反查（主路径）
Semrush / Ahrefs 拉竞品「活跃外链」，按博客 / 论坛 / 导航站 / .edu 分类导出。某游戏站案例 73% 外链来自博客评论，可直接复用。（帖 15 / 16）

### 2. 链轮挖掘法
A 站给 B 做了外链 → 继续抓 B 的外链 → 再抓下一层，无限递进。一个种子能滚出一大片。（帖 15 / 16）

### 3. 谷歌高级搜索 dork（直接挖提交站）
- 找列表站：`"submit your company" OR "submit company" OR "add your startup" site:.com` —— 实测能提交且是 dofollow。（帖 17）
- 找博客评论位：`"leave a reply" "your email" "<行业关键词>"`。（帖 05）
- 找相关文章：`site:` 语法找跟你主题相关的文章去评论。（帖 20）

### 4. 外链聚合平台（现成清单，不用自己挖）
- **[Backlink Dirs](https://backlinkdirs.com)**：收录高质量导航站 + 产品发布平台，可筛 `free` / `exchange-backlink` / `one-time-purchase` 三标签，按 `dr-desc` / `mv-desc` 排序，点 Submit 一键直达提交页。（脚本抓取返回 403，需浏览器手动打开）
- **[AI Dirs](https://aidirs.org)**（DR 39 / MV 1k，免费）、[bestlinks.530.news](https://bestlinks.530.news)、[dang.ai](https://dang.ai)。
- **[best-of-ai/ai-directories](https://github.com/best-of-ai/ai-directories)**：GitHub awesome list，100+ 免费 AI 目录主清单。
- **哥飞导航站联盟**：18 个中文出海开发者互助导航站（Woy AI DR 71、Tap4 AI DR 64、DokeyAI DR 61、AI With Me DR 54… 多数免费、对新站友好），可作为 targets.yaml 之外的补充站源。（帖 24）

### 5. 互换 + 付费（预算有限时的优先级）
**免费 → 互换 → 付费**三档（帖 09，Brennan）。互换站和付费站都能从 Backlink Dirs 对应标签筛。社群内还有「**愿意提供外链的网站收集**」的付费互助池，需联系哥飞撮合。（帖 23）有人整理了 AI 导航站外链库管理表，附各站 Stripe **折扣码 15%~30%**。（帖 08，Polaris）

### 6. 高权重源专攻（进阶，见第五部分）
HARO / 过期域名 / Testimonial / Best List 等，专门拿 DR 80+ 的稀缺外链。

## 四、博客评论外链（最常用的免费战术）

### 怎么做不被当垃圾
博客评论外链严格说是给互联网造垃圾，但实践证明仍有效，所以「明知不对也会做」，问题就变成**怎么让评论不那么垃圾**：

1. 用 `site:` 语法找跟你主题相关的文章，**读文章**，基于内容留有价值的讨论。
2. **评论正文不必出现广告**，只要「姓名」或「个人主页」字段填上你的链接就行。
3. 多数博客需站长审核，**只有被公开的评论才是有效外链**。少数不开审核的站提交即公开，但少且难找，靠抄同行作业。（帖 20，哥飞）

### 超链格式（覆盖约 95% 站点，一站通常只支持一种）
- 简版（不支持超链时）：`关键词 网址`
- 经典：`<a href="网址" title="Title">锚文本</a>`
- edu 变体（部分 .edu 评论页强制此格式，不支持 com/net/org 外后缀）：`<a href="网址" rel="nofollow ugc">锚文本</a>`
- BBcode：`[url=网址]锚文本[/url]`
- Markdown（如 new.web.cafe 本身）：`[锚文本](网址)`
- 显式 dofollow：`<a href="..." rel="dofollow">关键词</a>`

（帖 16 / 15）

### AI 评论模板
肯定文章 30 字 + 补充观点 50 字 + 自然植入锚文本。语义连贯、避免审核。（帖 15）

### nofollow → Dofollow 的绕过技术
> **风险评估**：技术上是在评论框输入合法 HTML、利用 WP 一个正则 bug，法律和博客平台层面都成立，是一种增长手法。要点是发权重的裁判是 Google：其 link spam 政策不认「绕过 nofollow 得到的权重」，所以这类链接的权重 Google 可能不予计算。好处是风险可控——不连累主站、可逆、会随 WP 修复自然失效。适合作补充手段、小规模使用。

WordPress 评论默认给外链加 `nofollow`，但有个 15 站实测 38-60% 成功率的绕过法（帖 05，Yifei 易飞）：

- **原理**：WP `wp_rel_nofollow()` 的正则 `|<a (.+?)>|i` 没加 `/s`(DOTALL)，`.+?` 默认不跨换行。在 href 里塞一个真实换行符（charCode 10），整个 `<a>` 标签不被匹配，nofollow 加不上。
- **做法**：在评论正文 textarea（不是 URL 字段）写 `<a href="https://你的站.com/\n">锚文本</a>`。手敲直接回车；Playwright/Puppeteer 用 `pressSequentially` 逐字敲（直接 set value 不触发 React onChange 会被反垃圾识别）。
- **验证 rel**：空字符串或 `external noopener noreferrer` = dofollow 成功；`ugc` 或 `nofollow` = 失败（**别被「发出去了」骗，ugc 一样不传权重**）。返回 `#comment-XXX` = 秒过，`?unapproved=XXX` = 进审核。
- **5 种失败模式**：wp_kses 直接 strip `<a>`、额外 sanitize 插件、hCaptcha Enterprise、Jetpack Highlander（跨域 iframe）、CleanTalk（403 硬拦）。
- 这是窗口期手法，官方加一个 `/s` flag 就失效。用的人越多死得越快。

## 五、进阶战术（拿高权重稀缺外链）

主要来自帖 10（宁神与 Gemini 的讨论）和帖 07（宗源 YouMind 增长复盘），密度很高。

> 这一节大多数 Google 认可、无政策风险（先创造价值、链接是回报）。**301 过期域名** 一条风险评估等级较高（Google 有专门条款 + 需买域名），下面 🔴 标注，按 ROI 自行取舍。🔴 = 风险等级标记，不是劝阻。

| 战术                           | 难度/效果                    | 怎么做                                                                                                            | 风险                |
| ---------------------------- | ------------------------ | -------------------------------------------------------------------------------------------------------------- | ----------------- |
| **HARO / Connectively**      | 平民拿 DR 80-90+ 的唯一途径      | 注册订阅记者求助邮件，1-2h 内回复、给可直接 copy 的干货、包装成「Founder of X」                                                            | 竞争激烈、成功率低、冷门行业请求少 |

| **Image Reclamation**        | 成功率极高                    | 做带数据的图表/信息图加 Alt，被 Google 图片收录后反向图搜找盗用者，礼貌要 source link                                                        | 无                 |

| **被动外链**                     | 长期                       | 写易被引用的统计数据类（「2024 X 消费数据统计」）、工具类内容                                                                             | 无                 |

### 增长侧的外链玩法（帖 07，YouMind 实战）
- **集合站打法**：做「免费而丰富的高价值资源」集合站（如提示词库），引发 KOL/自媒体自发传播。三月拿 50%+ 整站流量、数十个 dofollow 域名、DR +10。必备：快（模型发布头一周）、多语言（做了 16 语言，土耳其 KOC 单帖曝光 80w）、持续更新、UI 有特色。
- **GitHub 仓库引流**：集合站给仓库刷 stars → 被博主抓 trending → 自发推广；**超过 200 stars 能带来 DR 96 的 dofollow 外链**。
- **Product Hunt**：主要价值是**外链本身**（社区流量很少）。核心是备票 + 备 Hunter。只推 3 个 Top Hunter：Chris Messina（[chrismessina.me/product-hunt/coaching](https://chrismessina.me/product-hunt/coaching)，自称 hunt 100% 被 featured）、[benlang.me](https://benlang.me)、[x.com/rrhoover](https://x.com/rrhoover)。没被 featured 立刻下架改时间。
- **Chrome 商店蹭词**：标题放最想蹭的品牌词 + 信息词，分散到多语言。先申请 chrome featured badge（约 1000 用户时申请、约一周过审）再埋词。蹭词有被投诉风险（MyMind 被投诉后搜不到）。

### 自己做站给自己加外链？
帖 28 标题即问题，正文本轮未抓取成功。结合帖 04 的机制可推断哥飞的态度：自建站互链属于 built links，权重远低于 earned links，且容易形成可识别 footprint（同 IP / 同注册信息 / 互链网络），收益低风险高，不如把垂直站 outreach 做扎实。（**待补抓原文确认**）

## 七、风险红线 + 关键阈值

### 红线（不要做）
- ❌ **短期外链暴增** = link scheme，触发惩罚（帖 04）
- ❌ **锚文本单一化** = 触发 Google Penguin（帖 15）
- ❌ 买 Fiverr / 黑帽「高质量 backlink」服务，Google 认得这种 footprint
- ❌ 无关语种 PBN（土耳其/俄/越南语等），大概率判低质降权
- ❌ emoji 标题的「domain stats / URL shared」垃圾站，全 nofollow 零价值
- ⚠️ nofollow→dofollow 绕过、过期域名 301 是灰帽窗口期手法，别规模化
- ⚠️ 引流用户必须与产品强关联。反例 Wordware（明星转发月访 1000 万，次月跌 95%，700 万访客只注册 3-4%）（帖 07）

### 关键阈值（背下来）
| 指标 | 健康值 |
|---|---|
| 单日外链 | < 5 条（帖 15） |
| 外链月增长率 | 8-15%（帖 15） |
| 锚文本 LSI 占比 | > 40%（示例配比 50% 主词 / 各 10-15% LSI / 10% 品牌词或网址）（帖 15） |
| 域名筛选信号 | 搜索结果前 3 + 近 5 年注册（帖 19） |
| Semrush AS 值过滤 | 低于 90 的先过滤掉再关注（帖 16） |
| 目标站流量 | SimilarWeb 月访 1k+ 为佳（帖 15） |
| SEO 起量周期 | 约 1.5-2 个月（帖 01） |
| 时间分配（哥飞） | 挖需求 40% / 开发 20% / 推广 40%（帖 07） |

> 总原则：**外链虽好，切勿贪多**。Google 排名是多因素黑盒，on-page 内容质量不到位，堆外链也排不上去。（帖 19 / 04，反例 crazygamesonline.com 外链多但流量差）

### 被 K 后用外链恢复（帖 14）
内容不删不改几乎不新增；从减少收录起开始上外链；外链尽量大型站、多种文字组合、配图片外链。**最关键**：上几个新页面 → 直接给新页面做外链拉动全站再收录；给主页做外链时**不写主页核心词，而写新页面的核心词**，新页面再内链到别的页面，完成收录链条。
